# ServeRest API Automation Framework

## 📌 Descripción
Framework de automatización de pruebas API para ServeRest usando Karate DSL.

## 🚀 Tecnologías
- Java 11
- Maven
- Karate DSL
- JUnit 5

## 📂 Estructura del proyecto
src/test/java/runners -> TestRunner  
src/test/resources/features -> Features API  

## ▶️ Ejecución

```bash
mvn clean test
